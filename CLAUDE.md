# CLAUDE.md

## Design Context

This project has impeccable design context. **Read [PRODUCT.md](PRODUCT.md) (strategy) and [DESIGN.md](DESIGN.md) (visual system) before any UI work.**

- **Register:** `product` — design serves the task (dashboard, editor, scheduling, collaborators, settings). The root landing page is the one `brand` surface.
- **Users:** nervous, first-time speech-writers (best man, maid of honor, parent) plus invited collaborators. The job: turn nerves and a blank page into a finished speech, written *together*.
- **Personality:** warm, reassuring, celebratory — the supportive friend at the celebration table.
- **Design principles:** (1) calm the nerves first; (2) help people write, don't write for them; (3) better together; (4) warmth is structural, not decorative; (5) reveal complexity progressively.
- **Anti-references:** cold corporate SaaS · clinical AI writing tool · cluttered/overwhelming.
- **Visual system:** Deep Teal (#166162) on Warm Paper (#F6F5EF); Newsreader serif for emotion + Inter for UI; soft rounded surfaces, gentle ambient shadows; Celebration Gold reserved for real milestones. Full token spec in DESIGN.md / `.impeccable/design.json`.
